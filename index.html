Virus

Un virus informático es un software malicioso diseñado para infectar y dañar equipos electrónicos, como computadoras, teléfonos inteligentes y tablets.

Capacidad de replicarse: Los virus pueden crear copias de sí mismos y propagarse a otros equipos.
Se pueden propagar a través de redes, dispositivos USB, correos electrónicos o sitios web infectados.

GUSANOS INFORMATICOS !

Un gusano informático es un tipo de malware que se propaga a través de redes sin necesidad de infectar otros archivos. Se replica a sí mismo y utiliza las redes para saltar de un equipo a otro.

TROYANOS

Un troyano es un tipo de malware que se disfraza de software legítimo para engañar a los usuarios y obtener acceso a sus equipos. A diferencia de los virus y gusanos, los troyanos no se replican por sí mismos.

BOTS

Un bot es un programa informático que realiza tareas automáticamente. Los bots maliciosos, también conocidos como bots malintencionados, son bots que se utilizan para realizar actividades dañinas en Internet

KEYLOGGER

  • Ipsum

Un keylogger es un tipo de software o hardware malicioso diseñado para registrar secretamente las pulsaciones del teclado de un dispositivo infectado. Funciona como un espía silencioso, capturando cada tecla que se presiona y almacenando esa información para su posterior recuperación.

"ARCHIVOS BAT"

Un archivo BAT, o archivo de proceso por lotes, es un script que contiene una serie de comandos que se ejecutan secuencialmente en el sistema operativo Windows. Si bien son herramientas útiles para automatizar tareas, pueden ser utilizados con fines maliciosos para infectar equipos y comprometer la seguridad.
